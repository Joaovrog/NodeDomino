# NodeDomino

NodeDomino  é jogo de dominó online feito em html5,node.js e websocket.


## Getting Started

Acesse a pasta root do projeto e digite este comando para instalar:

```bash
npm install 
```


Depois digite esse comando para inciar aplicação:

```bash
npm start 
```
